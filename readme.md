### 清理评论.js for Codemao

顾名思义,按照正则表达式清理自动评论

#### [发布页](https://shequ.codemao.cn/work/231786149)

#### 使用

安装 Node.js 18.0.0+，因为本项目依赖的库需要 fetch API

然后输入
```bash
node main.js <你Cookie中的authorization项的值> <作品ID 或者 --user=你的训练师编号>
```
以运行评论清理

提示：可以修改 main.js 的代码以应用任意正则表达式规则
例如：
```JavaScript
// ...
// 下面这条将匹配 以“[任意字符]自动”开头的评论
const reg = /^.?自动.*/
// ...
```
